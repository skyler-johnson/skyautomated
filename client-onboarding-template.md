## Step 3: Kickoff Call Checklist

- [ ] Review their answers from the form
- [ ] Confirm which package they're on
- [ ] Set clear deadline: "I'll have a first draft to you by [DATE]"
- [ ] Get login access to any existing accounts (domain, social, etc.)
- [ ] Confirm payment method is set up
- [ ] Set revision expectations: "2 rounds of revisions included"

---

## Step 4: Delivery Checklist

Before sending the finished site to the client:

- [ ] Mobile responsive (test on phone)
- [ ] Contact form working (test it yourself)
- [ ] All links working
- [ ] Google Business Profile linked
- [ ] SSL certificate active (https)
- [ ] Page loads in under 3 seconds
- [ ] Spelling/grammar checked

## Notes
- Keep all client passwords in a secure password manager (1Password, Bitwarden)
- Never store client credit card info
- Document every change made to a client's site
- If something breaks, fix it BEFORE telling the client you're working on it
